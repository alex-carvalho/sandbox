# Ansible POCs

List of POCs for Ansible using diferente features

| POC | Description |
|---|---|
| [basic](basic/) | Minimal Ansible setup that installs and configures nginx on Docker containers |
| [variables](variables/) | Using variables, `host_vars`, and template rendering |
| [facts](facts/) | Ansible facts collected |
| [handlers](handlers/) | Handler usage for indepotence |
| [roles](roles/) | How to use roles to organize and reuse code |
| [vault](vault/) | Encrypts secrets with Ansible Vault |
| [collections](collections/) | How to use a community collection |
| [k8s](k8s/) | Manages Kubernetes resources via `kubernetes.core`  |
| [blue-green](blue-green/) | Simulates blue-green deployment switches an nginx proxy between two app slots |
